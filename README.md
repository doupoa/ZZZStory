你好绳匠，本项目旨在为剧情爱好者及资深游戏玩家提供完整的故事剧情，为非盈利公共项目。

本项目绝大部分内容均来自《绝区零》游戏及其官网，并将其制作成文档网后开源。

很高兴你能对该项目感兴趣，当然也非常希望您能参与项目共同补充完整的《绝区零》剧情故事。
